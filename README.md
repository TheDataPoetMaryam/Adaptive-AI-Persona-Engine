# 🤖 Synthetic Persona Intelligence  
### Context-Aware Conversational AI System Using Large Language Models

Synthetic Persona Intelligence is an experimental Generative AI project focused on building adaptive AI personas using Large Language Models (LLMs).

The system explores how prompt engineering, persona configuration, and conversational memory can be combined to create context-aware AI interactions.

The objective is not only generating responses, but designing AI behavior that remains consistent with assigned personality traits and conversation context.

---

## 🌟 Project Overview

Traditional chat systems generate responses based only on user input.  
This project introduces a configurable persona layer where the AI response style can be dynamically modified using structured prompts.

The system allows:

- Selection of different AI behavior modes
- Scenario-based interaction setup
- Context preservation using conversation history
- Dynamic response generation through LLM APIs

## 🛠️ Methodology

### 🎭 Persona Configuration

Multiple AI behavior styles are created using customized system instructions.

The persona layer controls:

- Response tone
- Interaction style
- Behavioral consistency

---

### 🧠 Conversation Memory

Previous interactions are stored using JSON-based memory management.

This allows the system to:

- Maintain conversation context
- Preserve previous exchanges
- Generate more consistent responses

---

### 🤖 LLM Response Engine

The system connects with a Large Language Model through API integration.

Workflow:

1. User sends input  
2. Persona instructions are combined with context  
3. Request is sent to the LLM  
4. Generated response is returned  
5. Conversation history is updated
